# 📡 Coverage Optimizer Pro

An intelligent spatial planning tool to optimize the placement of sensors (Cameras, Antennas) on 2D maps.

## 🏗️ Project Structure
- `app.py`: Main entry point.
- `src/logic/`: Simulation and mathematical masks.
- `src/ui/`: Gradio interface and styling.
- `assets/`: Static resources.

## 🚀 Getting Started
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `python app.py`

## 🛠️ Features
- Monte-Carlo optimization engine.
- Interactive Map Editor with layers.
- Multiple sensor profiles (Omni, FOV, Antenna).
- Precise coordinate exports in meters.
